# Mern auth template

# Installing Dep
    npm i
    cd frontend
    npm i

# Running App
    npm run dev
    
# Running Backend
    npm run server

# Running Frontend 
    npm run client

# Adding environment variables

Create .env file and add following environment variables in it.
    
    NODE_ENV , PORT , MONGO_URI , JWT_SECRET
